# 插画生成提示词模板

## 使用变量

- `{主题}`：要表达的项目、产品或文章主题
- `{核心机制}`：需要被视觉化的关键流程
- `{主体隐喻}`：机械装置、档案系统、卡片工作台等物理隐喻
- `{输出尺寸}`：默认 `1600 × 900`

## 主提示词

```text
为“{主题}”创作一张横版编辑插画，将“{核心机制}”转译为一个可触摸的“{主体隐喻}”。

Claude Cream Editorial Workshop 风格：暖象牙纸张与轻微灰泥质感背景，主体位于画面中右侧，左侧约 38% 保持干净留白。主体由哑光深炭黑金属、旧纸卡片、少量黄铜结构组成，用克制的琥珀金细线表现路径、连接和关键节点，仅用极少青绿色表示状态。

采用轻微俯视或平视三分之四视角，像精心制作的微缩模型和编辑类产品摄影。柔和的暖色自然侧光，阴影真实、边缘柔软，材质细腻，带轻微模拟胶片颗粒。画面安静、理性、有触感，不追求未来主义炫技。

底部约 20% 加入自然的深色渐层，为浅色标题叠加保留可读区域。主体完整，不在边缘截断关键结构。画面中不要出现标题、Logo、水印、可读 UI 文案或随机字符。

输出尺寸：{输出尺寸}。
```

## 负面约束

```text
避免冷蓝紫霓虹、赛博朋克、玻璃拟态、悬浮全息屏、通用人形机器人、扁平企业矢量插画、高饱和配色、拥挤满版构图、过度锐化、塑料质感、随机文字、乱码、Logo 和水印。
```

## 使用建议

1. 优先先确定一个具体物理隐喻，不要只输入抽象技术名词。
2. 每张图只表达一个核心机制，避免同时堆叠多个产品功能。
3. 生成时可将 `README.md` 中的参考封面作为视觉参考图一并提供。
4. 标题和品牌文字应在网站排版层添加，不让图像模型直接生成。
